# ftry

CLI minimale nommee `ftry`.

## Commandes disponibles

Les commandes sont pour l'instant des mocks. Elles affichent simplement leur nom dans le terminal.

- `ftry build`
- `ftry break`
- `ftry pop`
- `ftry land`

Exemple:

```powershell
ftry build
```

Sortie:

```text
build
```

## Installation locale

Prerequis:

- Python 3.10 ou plus recent
- `pip`

Depuis la racine du projet, installer la CLI en mode local editable:

```powershell
python -m pip install -e .
```

Ensuite, la commande est disponible dans le terminal:

```powershell
ftry break
```
